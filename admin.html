Connect your GitHub account so the admin panel can save project data and upload files directly to your repository.

How to get a PAT:
Go to github.com/settings/tokens → Generate new token (classic) → Select repo scope → Copy the token.

Your token is stored only in this browser session and never sent anywhere except GitHub's API.
